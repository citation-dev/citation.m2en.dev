# citation.m2en.dev

👀 Website & Document for citation

Build with [VuePress v2](https://v2.vuepress.vuejs.org/)

Deployment is [Cloudflare Pages](https://pages.cloudflare.com/)

## 開発サーバーの立ち上げ

```sh 
yarn dev
```

## ビルド

```sh 
yarn build
```
